# Sorteio de Amigo Secreto

Esse projeto é uma aplicação simples para fazer sorteio de amigo secreto. Você pode adicionar amigos, fazer o sorteio e ver quem foi sorteado.

## Funcionalidades

- **Adicionar Amigos**: Você pode adicionar os nomes dos seus amigos à lista.
- **Exibir a Lista de Amigos**: Todos os amigos adicionados aparecem na tela.
- **Sortear Amigo Secreto**: Quando tiver pelo menos dois amigos, você pode fazer o sorteio.
- **Resultado**: O nome do amigo secreto sorteado é mostrado na tela.

## Como Usar

1. **Adicionar um amigo**: 
   - Digite o nome de um amigo e clique em "Adicionar".
   - O nome será adicionado à lista.

2. **Sortear**: 
   - Clique em "Sortear" para ver quem foi sorteado como amigo secreto.
   
3. **Regras**: 
   - Precisa ter pelo menos 2 amigos na lista para sortear.
   - Não pode ter nomes repetidos na lista.

## Como Funciona o Código

- **`amigos`**: Armazena os nomes dos amigos.
- **`exibeTextoTela`**: Mostra texto na tela em qualquer lugar que você escolher.
- **`exibirListaAmigos`**: Exibe os amigos que você já adicionou.
- **`adicionarAmigo`**: Adiciona um novo amigo à lista, mas verifica se o nome não é repetido.
- **`exibirResultado`**: Mostra o amigo secreto sorteado.
- **`limparCampo`**: Limpa o campo onde você digita o nome.
- **`sortearAmigo`**: Sorteia um nome aleatório da lista de amigos.
- **`limparListaAmigos`**: Limpa a lista de amigos e o resultado na tela.

## Como Rodar

1. Clone ou baixe esse repositório.
2. Abra o arquivo HTML em qualquer navegador.
3. Aí é só começar a adicionar amigos e fazer o sorteio!
